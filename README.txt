Step 1: Install Python
If you don't already have Python, download and install the latest version from the official website: python.org
Important: During installation, be sure to check the box that says "Add Python to PATH" so you can run Python from anywhere in the Command Prompt.

Step 2: Install Required Libraries
Open CMD (Command Prompt) or Terminal and run the following commands to install all the necessary libraries.
Install the main libraries:
pip install pandas numpy matplotlib scikit-learn
pip install river

Step 3: Run the Application
Open CMD or Terminal.
Use the cd (change directory) command to navigate to the folder where you saved the ITITIU19035_NguyenTienGiaNguyen.py file.
Example: cd C:\Users\MSI\Desktop\Datamine
Run the application using the following command:
python ITITIU19035_NguyenTienGiaNguyen.py
For example, you can use the dataset file provided in the folder.
